# Backend Portfolio Site

## 프로젝트 개요
Java와 Spring Boot 기반 백엔드 개발자를 목표로 하는 학생의 자기소개, 기술 스택, 프로젝트 경험을 보여주는 포트폴리오 사이트입니다.

## 주요 기능
- 자기소개 출력
- 기술 스택 표시
- 프로젝트 소개
- GitHub 저장소 정보 연동
- Contact 정보 표시

## 사용 기술
- Java
- Spring Boot
- Thymeleaf
- HTML
- CSS
- JavaScript
- GitHub API

## 학습 내용
- Spring Boot MVC 구조
- Thymeleaf 화면 출력
- GitHub API 연동
- 프론트엔드와 백엔드 연결
